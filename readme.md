# Options:

```
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -b, --base     Base                                        [number] [required]
  -l, --listar   Lista tabla en consola               [boolean] [default: false]
  -h, --hasta    limite de la tabla                       [number] [default: 10]
```
